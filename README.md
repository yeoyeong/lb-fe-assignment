## 프로젝트 초기화 및 실행

### 1. **필수 조건**
- **Node.js**: LTS 버전(16 이상 권장)
- **npm**: Node.js 설치 시 포함됨

### 2. **프로젝트 복제**
```bash
git clone https://github.com/yeoyeong/lb-fe-assignment.git
cd lb-fe-assignment.git
```

### 2. **의존성 설치**
```bash
npm install
```

### 3. **서버 실행**
```bash
npm run dev
```
